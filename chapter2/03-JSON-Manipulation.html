03-JSON-Manipulation

json是互联网中常用的数据交换格式,可以被大部分语言解析,python支持JSON格式解析输出,在前面的例子中,我们发送请求接收返回的JSON数据

  • 使用python解析JSON数据
import json

data = '{\n    "userId": 1,\n    "id": 1,\n    "title": "sunt aut facere repellat provident occaecati excepturi optio reprehenderit",\n    "body": "quia et suscipit\\nsuscipit recusandae consequuntur expedita et cum\\nreprehenderit molestiae ut ut quas totam\\nnostrum rerum est autem sunt rem eveniet architecto"\n  }'
parsed = json.loads(data)
type(parsed) # list

现在变量parsed被解析成一个可以被轻松解析的字典
