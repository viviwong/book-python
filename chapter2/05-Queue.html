05-Queue

多线程是一个很大的话题,它包含很多内容如:锁机制等等,对于实现网络爬取,还有一个重要的话题

  • 任何网站都建由互联的页面组成的,wikipedia的首页应该是这样的
---- front page (https://en.wikipedia.org/wiki/Main_Page)
|--- Leonard Cohen (https://en.wikipedia.org/wiki/Leonard_Cohen)
|--- Chicago Cubs (https://en.wikipedia.org/wiki/2016_Chicago_Cubs_season)
        |--- National League (https://en.wikipedia.org/wiki/National_League)

所以当你使用5个线程实现页面爬取的时候,如何给五个线程分配任务呢,那么就会有一个任务集(里面全是任务),还有一个资源池去消化这些任务,这类问题叫生产者消费者问题 ,解决这个问题我们通常使用 队列安全中的FIFO(先进先出)队列

  • FIFO Queue 先进先出队列 first in first out demo
from queue import Queue

# 新建一个队列对象
q = Queue()
for i in range(3):
    q.put(i)

# 查看队列和队列类型
print(q,type(q))

#查看第一个取的是几号
print(q.get())

while not q.empty():
    print(q.get())
  • 队列结合多线程使用
from threading import Thread
from queue import Queue

# 执行任务函数
def do(q,i):
    # 设置为True让线程随时接受任务
    # 为什么这里用True不用 not q.empty()
    # 因为线程一旦为空就会一直死循环 https://segmentfault.com/q/1010000007565364/a-1020000007566927
    while True:
        print('线程%d执行任务%d' % (i,q.get()))
        # 相当于每次执行完get后task_done() 从队列中删除一个元素
        q.task_done()

# 初始化队列和线程        
q = Queue()
num = 5

# 线程 人
for i in range(num):
    worker = Thread(target=do,args =(q,i))
    worker.start()

# 队列 任务   
for x in range(100):
    q.put(x)

# join() 根据队列长度是否为零判断是否结束线程
q.join()
  • 队列结合多线程模拟页面多层爬取
from queue import Queue
import random
import threading
import time

# 初始化队列 队列计数 线程锁
q = Queue()
counter = 0
lock = threading.RLock()

# 执行任务
def worker(i, q):
    while True:
        # 执行队列中的任务
        task = q.get()
        print('%d working on task %d' % (i, task))
        time.sleep(1)
        # 往队列中添加任务
        count_new_tasks = random.randrange(0, 10)
        print('putting %d new tasks in queue' % count_new_tasks)
        for new_task in range(count_new_tasks):
            q.put(new_task)
        q.task_done()
        # 数数 
        global counter
        with lock:
            counter += 1
            print('processed %d tasks so far' % counter)

# 创建线程
for i in range(2):
    t = threading.Thread(target=worker, args=(i, q, ))
    t.setDaemon(False)
    t.start()

q.put(1)

q.join()
