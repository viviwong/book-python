01-List-Comprehension

在之前我们使用了python内建的数据结构 list,tuple,dictionary
当你需要存放一组数据(或者做数据收集)的时候非常有用
你经常需要对集合进行操作,例如从集合中过滤出元素

  • 从original中过滤出小于3的数
result = []
original = [1, 2, 3, 4, 5]
for i in original:
    if i < 3:
        result.append(i)
result,type(result)
  • python提供了一种更好的方式达成上面的效果列表推导式,你可以通过一行实现上面的效果
original = [1, 2, 3, 4, 5]
result = [x for x in original if x < 3]
result,type(result)
  • 列表推导式同样可以用于生成集合
original = [1, 2, 3, 4, 5]
result = {x for x in original if x < 3}
  • 列表推导式同样可以用于生成字典
original = [1, 2, 3, 4, 5]
result = {x:x for x in original if x < 3}
result,type(result)
  • 列表推导式同样可以用于多个变量
  • 获得letters中不是a,numbers中数字小于3的元祖组合
letters = ['a', 'b', 'c', 'd']
numbers = [1, 2, 3, 4, 5]
result = [(x,y) for x in letters if x is not 'a' for y in numbers if y<3]
result,type(result[0])
  • 生成器是列表推导式的一种特殊情况,回到从列表中找出小于3的数,使用列表推导式一次性显示所有结果
original = [1, 2, 3, 4, 5]
result = [x for x in original if x < 3]
  • 但是如果是一个大的数据集合,最好使用生成器
original = range(100)
gen = (x for x in original if x < 50)
type(gen)
# python2使用
gen.next()
gen.next()

# python3使用
gen.__next__()
gen.__next__()

1 首先gen变量现在的类型是生成器而不是列表
2 使用生成器你可以通过next()(python2) __next__()(python3)方法遍历,使你能够处理海量数据,而不会对内存产生过大压力

  • 现在我们从句子中过滤出非元音字母,我们使用生成器的方式而不是列表推导式
vowels = 'aeiou'
sentence = 'Finally, you can try to quickly find non-vowel characters in a sentence.'
# 列表推导式
nonvowels1 = "".join([x for x in sentence if x not in vowels])
nonvowels1,type(nonvowels1)

# 生成器
nonvowels2 = "".join((x for x in sentence if x not in vowels))
nonvowels2,type(nonvowels2)
