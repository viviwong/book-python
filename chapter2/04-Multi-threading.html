04-Multi-threading

  • 我们发送HTTP请求获得JSON相响应,我们可以发送100次请求
import http.client

def get_data():
    conn = http.client.HTTPSConnection('jsonplaceholder.typicode.com')
    conn.request('GET', '/posts/1')
    resp = conn.getresponse()
    resp.read()

for i in range(100):
    get_data()
    print(i)

我们之前一直使用单线程在处理程序,你可以认为是一个工作单元,现代计算机中,你可以利用更多线程执行工作,提高速度

#coding=UTF-8
import http.client
import threading

def worker(n):
    for i in range(10):
        conn = http.client.HTTPSConnection('jsonplaceholder.typicode.com')
        conn.request('GET','/posts/1')
        resp = conn.getresponse()
        data = resp.read()
        print("线程"+str(n)+"=>第"+str(i)+"个任务")

threads = []
# help(threading)
for i in range(5):
    # 初始化创建线程,t的类型是线程
    t = threading.Thread(target = worker,args=(i,))
    print("========"+str(i)+"=========")
    print(t,type(t))
    # 线程池,threads类型为列表
    threads.append(t)
    print(threads,type(threads))
    # 启动创建的线程
    t.start()
    print("========线程启动了=======")

1 导入threading创建5个线程 2 5个线程同时进行执行一个目标函数,可以给你提高5倍的速度

  • 可以看着这样,主线程下面有5个新线程,他们独立工作
--- main thread
 |- thread 1
 |- thread 2
 |- thread 3
 |- thread 4
 |- thread 5
  • 之前worker(10)函数中的参数是固定的,现在让worker(num)函数接受参数变得灵活一些
import http.client
from threading import Thread
import time

def worker(num):
    for i in range(num):
        time.sleep(1)
        conn = http.client.HTTPSConnection('jsonplaceholder.typicode.com')
        conn.request('GET', '/posts/1')
        resp = conn.getresponse()
        resp.read()

threads = []
for i in range(5):
    t = Thread(target=worker, args=(10, ))
    threads.append(t)
    t.start()

1 worker(num)函数带参数,通过args=(10,)传递参数
2 设置time.sleep(),降低对爬取服务器的压力
