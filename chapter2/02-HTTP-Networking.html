02-HTTP-Networking

作为一个出色的编程语言,python支持访问互联网,允许你在不打开浏览器的情况下获取数据

  • 接下来我们访问网站jsonplaceholder,可以实现随机输出的网站
import http.client
# 连接
conn = http.client.HTTPSConnection('jsonplaceholder.typicode.com')
# 请求
conn.request('GET', '/posts')
# 响应
resp = conn.getresponse()
# 读取
data = resp.read()

1 首先我们导入python内建的库httplib(python2),http.client(python3)
2 因为typeicode网站使用HTTPS编码,所以我们使用HTTPSConnection创建连接
3 我们请求/posts地址从typeicode网站获得随机发布的数据
4 获得的响应,我们能以JSON格式检索数据
5 步骤 连接 请求 响应 读取
