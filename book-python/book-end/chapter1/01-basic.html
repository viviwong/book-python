3 数据结构

3.1 列表 List

普通的数据集合

  • 列表定义
a = []
b = ['ted', 'marshall', 'lily', 'barney', 'robin']
  • 修改列表的值
b = ['ted', 'has', 1, 'apple']
b[2] = 2
  • 列表切片
b[1:]
b[-1]
b[0:2]
b[0::2]
del b[0]
  • 列表合并和追加
a = ['ted', 'marshall']
b = ['barney', 'robin']
a + b
a.append('lily')

3.2 元祖 Tuple

不可以修改的列表

  • 定义元祖

a = ()
a = (1, 2, 3)
a = ('ted', 'has', 1, 'apple')
a[2]
  • 元祖切片
b = ('ted', 'has', 1, 'apple')
b[1:]
b[-1]
b[0:2]
b[0::2]
  • 元祖中元素的值是不能修改的

b[0] = 'this will not work'

5 集合 set

集合中的元素不重复,集合不能通过下标访问,需要转换为列表后才能访问

  • 定义集合
a = set()
b = set([])
c = set([1, 2, 2, 3, 4])
  • 集合元素访问
a = set([1, 2, 3, 4, 4, 5])
a[0]    # - this will blow up
list(a)[0]
  • 集合可以参与逻辑运算
a = set([1, 2, 3, 4])
b = set([2, 3, 4, 5])
a | b
a & b

4 流程控制

分支结构&循环结构

4.1 分支结构

a = 1
if a == 1:
    print('value of a is 1')
elif a > 1:
    print('value of a is bigger than 1')
else:
    print('value of a is smaller than 1')

4.2 循环结构

  • for-loop
# - print ITecho 10 times
for i in range(10):
    print('ITecho %d' % i) #%d占位符 做格式化输出 实际输出为i的值
  • while-loop
a = 10
while a > 0:
    print('bittiger')
    a -= 1

5 函数 function

我们使用函数将多个代码逻辑组合在一起,调用函数实现复用

  • 不带参数的函数
# - define a function called print_vivi
def print_vivi():
    for i in range(10):
        print('vivi %d' % i) # i是局部变量

print_vivi()
print_vivi()
  • 带参数的函数
def print_vivi(round):
    for i in range(round):
        print('vivi %d' % i)

print_vivi(1)
print_vivi(100)
  • 将函数返回值传递给变量
def get_course():
    return 'vivi'

course = get_course()
  • 一次返回多个值

def get_course():
    return 'vivi', 'python'

course = get_course()   # - course will be ('vivi', 'python')
school, course = get_course()   # - school will be 'vivi', and course will be 'python'
