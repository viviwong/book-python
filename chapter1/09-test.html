09-test

python中级

09-test0.1

没有测试程序是不完整的
我们一直在手动测试代码
对于大型项目并不适用
作为开发者,我们需要在代码中自动执行测试操作
用于测试代码的片段叫测试用例
python中有很多框架帮助你写测试用例

  • 以pytest为例写测试用例,新建文件test_schedule.py
# coding=UTF-8
# 测试的代码段叫测试用例

# assert段言
# 表示必须满足的条件,例如下面触发添加任务后,我们断言test必须在s.tasks列表中

from schedule import Schedule

# 测试用例是python代码
# 测试函数必须以test开头
def test_add():
    s = Schedule('s1','cs')
    s.add_task('test','test_content',100)
    assert 'test' in s.tasks

def test_remove():
    s = Schedule('s1','cs')
    s.remove_task('test')
    assert s.tasks == {}

注意文件名必须是test_+被测试文件名 注意测试文件函数名必须是test_方法名

  • 把他和schedule.py文件放到同一目录,目录结构如下
.
├── schedule.py
└── test_schedule.py
  • 测试用例定义好后,执行测试
