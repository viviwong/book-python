02-module

module, package, building block 这些都属于组件 模块可以让你重复使用一段代码

  • 创建文件,文件名为calculator.py
  • 现在就拥有一个模块叫calculator
def add(a, b):
    if type(a) is int and type(b) is int:
        print(a + b)
    else:
        print('inputs are not int type')
  • 模块可以在不同的地方重复被使用
  • 在同一个文件件创建文件test.py
import calculator

num1 = input('input first number: ')
num2 = input('input second number: ')
calculator.add(num1, num2)
  • 命令行中执行python程序
python test.py

1 我们导入了calculator模块,我们可以使用模块中的成员如calculator.add()函数
2 你可以直接导入模块里面的方法add,如from calculator import add,这样你可以直接使用add()函数,不用在开始的时候导入calculator
