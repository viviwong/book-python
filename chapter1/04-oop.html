04-oop

回顾程序我们可以看到,程序就是数据和对数据的操作逻辑

  • 现在你需要存储和检索任务数据
    • 任务数据使用存字符串存储
    • 任务数据集合使用列表 元祖或集合存储
# coding=UTF-8
# 脚本中有中文使用

# 字符串存储任务数据 
s1 = 'work schedule'
s2 = 'person schedule'
s3 = 'play schedule'

# 用元祖 列表 集合存储任务数据集合
tasks = [s1,s2,s3]

print(tasks)
print(s1,type(s1))
print(s2,type(s2))
print(s3,type(s3))

1 这种存储方式应对简单情况是可以的 2 但是如果要添加新属性,旧的代码无法处理

  • 因此存储数据的时候最好带一些结构,这就是OOP
class Schedule(object):

    name = None
    desp = None

    def __init__(self, n, d):
        self.name = n
        self.desp = d

    def __str__(self):
        return 'Schedule %s, %s' % (self.name, self.desp)

1 创建一个类叫Schedule,类好比建筑的蓝图,而不是建筑本身,他包含怎么去建造他 2 Python中每个类都有内置的方法,如__init__告诉你如何将类初始化为对象 3 __str__方法可以获得对象中可读的信息 4 name和desp是类Schedule的成员变量

  • 初始化类为对象 
# 实例化类 
s1 = Schedule('s1', 'work schedule')
s2 = Schedule('s2', 'personal schedule')
s3 = Schedule('s3', 'play schedule')

print(s1,type(s1))
print(s2,type(s2))
print(s3,type(s3))

print(s1.desp)
print(s2.desp)
print(s3.desp)

print(s1)
print(s2)
print(s3)

1 通过类创建了三个对象 s1,s2,s3 2 打印对象自动触发__str__执行 3 self是一个特定的变量,他指向当前对象自己

  • 对于类我们可以通过增加方法来增加他的功能
class Schedule(object):

    name = None
    desp = None
    tasks = None

    def __init__(self, n, d):
        self.name = n
        self.desp = d
        self.tasks = {}

    def add_task(self, name, content, priority):
        if name not in self.tasks:
            task = [name, priority, content]
            self.tasks.update({name: task})
        self.display()

    def remove_task(self, name):
        if name in self.tasks:
            self.tasks.pop(name, None)
        self.display()

    def display(self):
        if self.tasks:
            print('here are your tasks:')
            for v in self.tasks.values():
                print('%-10s %-5s %-50s' % (v[0], v[2], v[1]))

    def __str__(self):
        return 'Schedule %s, %s' % (self.name, self.desp)

# 实例化类
s1 = Schedule('s1','work schedule')
s2 = Schedule('s2','personal schedule')
s3 = Schedule('s3','play schedule')


print(s1,type(s1))
print(s2,type(s2))
print(s3,type(s3))

print(s1.desp)
print(s2.desp)
print(s3.desp)

以上类具有了添加和删除任务的功能了
