07-exp

读写文件是非常危险的事情,比如文件不存在,或者文件崩溃,这些都会导致python文件异常退出,python提供了机制去定位和处理这些异常

  • 异常处理
# 异常捕获
try:
    # - awesome code goes here
    # 可能出现异常的代码
except Exception as e:
    # - do something about the exception
    # 异常处理
finally:
    # - do final clean up
    # 清理

1 finally 代码块无论如何都会执行,对代码进行清理 2 Exception 是python中的一个基类,可以代表所有异常

  • 自定义异常
class AwesomeException(Exception):

    def __init__(self, *args, **kwargs):
        Exception.__init__(self, *args, **kwargs)

raise AwesomeException('somehow, this is an awesome exception')

1 我们通过异常类创建异常 2 python中异常也是一个对象 3 AwesomeException是子类,继承父类Exception,他具有父类所有的属性 4 你可以使用raise 触发一个异常
