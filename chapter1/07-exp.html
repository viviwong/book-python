07-exp

读写文件是非常危险的事情,比如文件不存在,或者文件崩溃
这些都会导致python文件异常退出
python提供了机制去定位和处理这些异常

异常处理 try-except

  • 异常捕获demo
# 异常捕获
try:
    # - awesome code goes here
    # 可能出现异常的代码
except Exception as e:
    # - do something about the exception
    # 异常处理
finally:
    # - do final clean up
    # 清理

1 finally 代码块无论如何都会执行,对代码进行清理
2 Exception 是python中的一个基类,可以代表所有异常

  • 抛出异常并且捕获
# coding=UTF-8

try:
    # 产生异常代码 文件不存在
    with open("../haha.txt","r") as f:
    # 正常代码
#    with open("../test.txt","r") as f:
        print(f.read())
except Exception as e:
    print("文件不存在")
    # 异常对象内容
    print(e)
# finally里面的内容无论如何都执行 
finally:
    print("我执行了哦")

自定义异常

  • 自定义异常demo
class AwesomeException(Exception):

    def __init__(self, *args, **kwargs):
        Exception.__init__(self, *args, **kwargs)

raise AwesomeException('somehow, this is an awesome exception')

1 我们通过异常类创建异常
2 python中异常也是一个对象
3 AwesomeException是子类,继承父类Exception,他具有父类所有的属性
4 你可以使用raise 触发一个异常

  • 创建自定义异常并且捕获
#自定义异常类
class AwesomeException(Exception):
    def __init__(self, *args, **kwargs):
        Exception.__init__(self, *args, **kwargs)

# 触发异常
# raise AwesomeException('somehow, this is an awesome exception')

# 使用try-except处理异常
try:
    raise AwesomeException('somehow, this is an awesome exception')
except AwesomeException as ae:
    print("产生异常了")
    # 打印异常对象内容
    print(ae)
