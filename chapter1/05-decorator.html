05-decorator

这种里面包含函数,然后修改他使用前和使用后的结果就叫装饰器

  • 任务添加和删除的时间
# 任务+描述 加上功能 可以添加任务内容和优先级 添加时间显示

from datetime import datetime

# 添加时间显示函数
def display_time():
    print(str(datetime.now()))

class Schedule(object):
    name = None
    desp = None
    tasks = None

    # 1 类的初始化
    def __init__(self,n,d):
        self.name = n
        self.desp = d
        self.tasks = {}

    # 2 打印对象输出内容
    def __str__(self):
        return "schdule1 %s %s" % (self.name,self.desp)

    # 3 添加任务
    def add_task(self,name,content,priority):
        if name not in self.tasks:
            task = [name,content,priority]
            self.tasks.update({name:task})
        self.display()
        #添加时间显示
        display_time()

    # 4 删除任务
    def remove_task(self,name):
        if name in self.tasks:
            del self.tasks[name]
        self.display()
        # 删除时间显示
        display_time()

    # 5 显示对象中的任务
    def display(self):
        if self.tasks:
            print("your tasks is below: ")
            print("任务名   ,  优先级 , 任务内容")
            for v in self.tasks.values():
                print("%-10s %-5s %-50s" % (v[0],v[2],v[1]))
        else:
            print("your tasks is null")

#类初始化为对象
s6 = Schedule('任务2','任务2的描述')
print(s6)

#添加任务
s6.add_task("任务2","打豆豆",1)

# 删除任务
s6.remove_task("任务2")

python中函数是一级对象,可以将一个函数作为参数传递到另一个函数中

  • 函数也是一个对象,函数里面定义一个函数 
# coding=UTF-8

# 函数里面定义了一个函数 
# 因为函数是对象 所以函数里面可以定义函数;好比对象里面定义函数 
def outer():
    def inner():
        return 666
    # 函数outer的返回值是一个函数名
    return inner

# 将函数outer赋值给var
var = outer()  
# 打印函数名
print(outer)  # outer函数名对应的地址
# 打印函数
print(outer()) # inner函数的地址
# 打印函数的函数 
print(outer()()) # 666
# 打印变量名函数
print(var())   # 666

1 函数是一个对象,所有函数里面也可以嵌套函数
2 outer函数返回值是一个函数
3 你可以将函数传给一个变量并且调用这个变量
4 函数和字符串 数字是一样的使用 只不过函数后面多一个()

  • 装饰器演示
  • 从上面的例子我们可以看到
  • 1 函数里面可以嵌套函数
  • 2 函数的返回值可以是函数
  • 3 函数可以传递给变量,那么函数就可以作为参数传递给另一个函数了
  • 函数display_decorator传递参数func,里面定义一个函数wrapper,返回值是函数wrapper
# coding=UTF-8

def display_decorator(func):
    def wrapper():
        print('before func')
        func()
        print('after func')
    # 返回wrapper函数
    return wrapper

def test_function():
    print("this is awesome function")

# 直接使用函数 test_function
test_function()

# 通过display_decorator使用test_function
display_decorator(test_function)() # before t a

1 display_decorator就是装饰器
2 他里面包裹了一个函数 func
3 并且通过wrapper函数修改了 func函数使用前和使用后的内容
4 这个机制就是装饰器
这种里面包裹一个函数,然后修改他使用前和使用后的行为的机制就叫装饰器

  • python为装饰器的易用性提供的语法如下
# coding=UTF-8

# 这个函数是装饰器
def display_decorator(func):
    def wrapper():
        print('before func')
        func()
        print('after func')
    return wrapper

# 这里是使用装饰器
@display_decorator
def test_function():
    print("this is awesome function")

# 使用装饰器后函数的调用
test_function() # b t a
  • 任务添加时间的代码修改为
# coding=UTF-8

# 装饰器 
# 添加日期
from datetime import datetime

# display-decorator就是装饰器
def display_decorator(func):
    # *非关键字收集参数,**关键字收集参数 
    # func是装饰器作用的函数 add_task remove_task
    # 这些函数的参数个数和类型都不确定,所以使用收集参数
    # 在这个例子中可以只使用* 非关键词收集
    def wrapper(*args, **kwargs):
        print(str(datetime.now()))
        return func(*args, **kwargs)
    return wrapper


class Schedule(object):

    name = None
    desp = None
    tasks = None

    def __init__(self, n, d):
        self.name = n
        self.desp = d
        self.tasks = {}

    # @+装饰器 使用装饰器的快捷键
    @display_decorator
    def add_task(self, name, content, priority):
        if name not in self.tasks:
            task = [name, priority, content]
            self.tasks.update({name:task})
        self.display()

    @display_decorator
    def remove_task(self, name):
        if name in self.tasks:
            self.tasks.pop(name,None)
        self.display()

    def display(self):
        if self.tasks:
            print("here are your tasks:")
            for v in self.tasks.values():
                print("%-10s %-5s %-50s" % (v[0], v[2], v[1]))

    def  __str__(self):
        return "Schedule %s , %s" % (self.name,self.desp)


s = Schedule('s1','ITshcool')
# 对象s添加任务详情
s.add_task('cs','teach introduction to python course',1000)

# 对象s删除任务详情
s.remove_task('cs')

# 查看对象s任务
print(s)

@装饰器 放到函数或方法前面是使用装饰器修饰函数或方法的快捷键
*和** 是非关键词参数收集和关键词参数收集,用于从列表和字典中收集参数
